# 🔐 Cryptography Project – Interactive Encryption Lab

## 📌 Overview
This project is an interactive web-based cryptography laboratory that demonstrates multiple encryption and hashing algorithms in a visual and educational way. It is designed to help learners understand how classical and modern cryptographic techniques work through real-time interaction.

The project is built entirely using **HTML, CSS, and JavaScript** with no external frameworks.

---

## 🧠 Features

### 🔴 Classical Encryption
- Caesar Cipher (with brute force & frequency analysis)
- Vigenère Cipher (key-based encryption)
- ROT13 Cipher
- Atbash Cipher

### 📦 Encoding
- Base64 Encode / Decode
- URL-Safe Base64 Encoding

### ⚡ Stream Encryption
- XOR Cipher (encrypt/decrypt using same operation)

### 🔑 Hash Functions
- MD5 (educational implementation)
- SHA-1
- SHA-256
- SHA-512 (via Web Crypto API)

### 🛡️ Security Tools
- Password strength checker
- Hash comparison tool
- Data integrity verification

### 📡 Additional Features
- Morse Code conversion
- Algorithm comparison table
- Frequency analysis for cryptanalysis
- Interactive UI with tab system

---

## 🎯 Purpose
The goal of this project is to:
- Teach cryptography concepts in a practical way
- Demonstrate how encryption algorithms work internally
- Provide hands-on learning instead of theoretical explanation
- Simulate real cryptographic tools in the browser
